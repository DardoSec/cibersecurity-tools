# Cybersecurity Tools - Educational Scripts

## Description
A collection of friendly educational tools to explore basic cybersecurity concepts.
Includes historical ciphers, encoding/decoding techniques, and hands-on tools to learn core security principles.

---

## Contents
- Ciphers: Classical encryption and decryption tools.
- Encoding/Decoding: Convert data between formats like Base64, Hex, Morse, etc.
- OSINT Exercises: Basic open-source intelligence tools for learning purposes.

---

## Usage
Each folder contains scripts or executables with instructions.
To use a script, follow the README or instructions inside the respective folder.

## Goals
- Teach basic cybersecurity concepts in a practical way.
- Provide hands-on tools for cryptography and data encoding.
- Encourage experimentation and learning by doing.

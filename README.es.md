# Herramientas de Ciberseguridad - Scripts Educativos

## Descripción
Una colección de herramientas educativas amigables para explorar conceptos básicos de ciberseguridad.  
Incluye cifrados históricos, técnicas de codificación/decodificación y herramientas prácticas para aprender los principios fundamentales de seguridad.

---

## Contenidos
- Cifrados: Herramientas clásicas de encriptación y desencriptación.
- Codificación/Decodificación: Convertir datos entre formatos como Base64, Hex, Morse, etc.
- Ejercicios de OSINT: Herramientas básicas de inteligencia de fuentes abiertas con fines educativos.

---

## Uso
Cada carpeta contiene scripts o ejrcutables con instrucciones.  
Para usar un script o ejecutable, sigue el README o las instrucciones dentro de la carpeta correspondiente.

---

## Objetivos
- Enseñar conceptos básicos de ciberseguridad de forma práctica.
- Proporcionar herramientas prácticas para criptografía y codificación de datos.
- Fomentar la experimentación y el aprendizaje mediante la práctica.
